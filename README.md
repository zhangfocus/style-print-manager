# 款式印花管理系统

基于 FastAPI + SQLite 的印花贴图管理后台，支持 Excel 批量导入/导出。

## 功能模块

| 模块 | 说明 |
|------|------|
| 款式（Style）| 管理服装款式基础信息 |
| 印花（Print）| 管理印花图案信息 |
| 位置（Position）| 管理贴花位置信息 |
| 限定（Restriction）| 定义某款式+某位置允许使用哪些印花 |

## 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务（首次启动自动建表）
python run.py
```

浏览器访问：http://localhost:8000

## Excel 导入格式

下载模板：`GET /api/excel/template`

模板包含 4 个 Sheet：

### 款式 Sheet
| 款式编号* | 款式名称* | 品类 | 颜色 | 备注 |
|-----------|-----------|------|------|------|

### 印花 Sheet
| 印花编号* | 印花名称* | 图案类型 | 色系 | 备注 |
|-----------|-----------|----------|------|------|

### 位置 Sheet
| 位置编号* | 位置名称* | 区域 | 备注 |
|-----------|-----------|------|------|

### 限定 Sheet
| 款式编号* | 位置编号* | 印花编号* | 是否启用(1/0) | 备注 |
|-----------|-----------|-----------|---------------|------|

> `*` 为必填字段。相同编号的记录会自动更新（upsert），不会重复插入。

## API 文档

启动后访问：http://localhost:8000/docs

## 技术栈

- **后端**: Python 3.10+ / FastAPI / SQLAlchemy 2.0
- **数据库**: SQLite（文件 `data/db.sqlite3`）
- **Excel**: openpyxl
- **前端**: Bootstrap 5 + 原生 JS（无构建工具）
